ОПИСАНИЕ
Скрипт для сбора информации о посетителях. Не требует mysql,
собранная инфа хранится в текстовом файле.

УСТАНОВКА
Для установки скрипта киньте файлы userinfo.php, view.php,
visitors.txt на ваш сервер? установите права доступа 777
на файл visitors.txt (не для всех хостингов). Для того чтобы
скрипт собирал информацию о всех посетителях сайта, во все 
страницы нужно включить код: 
<?php include("http://yoursite.com/путь_к_папке_со_скриптом/userinfo.php"); ?>
Для просмотра пользователей зайдите по адресу:
http://yoursite.com/путь_к_папке_со_скриптом/view.php
