FULL STYLE
Update: 15-01-2018

Автор: Vitaliy V.
Браузер: SeaMonkey 2.46 +
ОС: Windows, Linux
Описание: Стиль для моего расширения Style CSS
Содержимое архива распаковать в папку профиля SeaMonkey.
Использовать данный стиль с темой по умолчанию или с лёгкими темами, если тема тёмная со светлым текстом
будет использоваться оформление от стиля (как на скриншотах) которое можно легко изменить
заменив три изображения для фона в папке png и в стиле подобрать нужный цвет, значением переменной --moz-lwtheme-brighttext-background
Содержимое файла stylecss_author.css почти не нужно для интерфейса и можно удалить.
Чтобы добавить кнопки закрытия на вкладки раскомментируйте код после комментария /*Кнопки закрытия на вкладках*/
или закрывать вкладки СКМ, но удобнее ПКМ Right click close tab.
Чтобы изменить расположение кнопки новой вкладки в окне почты раскомментируйте код после комментария /*Кнопка новой вкладки в окне почты*/ .
Для включения многострочной панели вкладок нужно подключить в настройках Style CSS доп. файл стиля multirowtabs.css
или скопировать из multirowtabs.css в основной стиль stylecss.css ...

Стиль не простой и не предназначен для всех...

Чтобы убрать / заменить заголовок окна, установите расширение Control Buttons





Боковая панель (если включена) появляется при наведении на левый край (2px в развёрнутом окне и 5px в обычном).
