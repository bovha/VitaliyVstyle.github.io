Adds Toolbar and Buttons Без перезапуска
Развернуть
Adds Toolbar and Buttons добавляет панель инструментов и кнопки.
Вы найдете кнопки, для управления скрытыми функциями или требующими слишком много кликов, но некоторые функции добавляет данное расширение.

Long Left Click Без перезапуска
Развернуть
Long Left Click открывает ссылки и закладки в новой вкладке "долгим" кликом.

Load Favicons Remove Duplicates Без перезапуска
Развернуть
Load Favicons Remove Duplicates восстанавливает иконки и удаляет дубликаты закладок

Agent Sheet Без перезапуска
Развернуть
Agent Sheet изменяет скроллбар и всплывающие подсказки.

Control Buttons
Развернуть
ОС: Windows, Linux.
Control Buttons добавляет кнопки (с дополнительными функциями) управления окном.
В SeaMonkey (Windows, Linux), Firefox (Linux), Pale Moon (Linux) скрывает / заменяет заголовок окна.

eCleaner Без перезапуска
Развернуть
eCleaner очищает настройки расширений оставленные после удаления, а также мусорных файлов в профиле и т. д.

Right click close tab Без перезапуска
Развернуть
Right click close tab позволяет закрывать вкладки правой кнопкой мыши. Для отмены удерживайте клавишу ( Shift | Ctrl ) + ПКМ чтобы появилось контекстное меню вкладки.

Flip Close Tab Без перезапуска
Развернуть
Flip Close Tab переключает фокус на последнюю посещённую вкладку при закрытии текущей или при клике по кнопке расширения.

Old search Без перезапуска
Развернуть
Не совместимо с Quick Search Bar (см. ниже).
Old Search возвращает старый способ поиска (в FF 43 и выше) для панели поиска.
И добавляет новые функции: индикатор добавления поиска, опция очистить форму после поиска и др.

Quick Search Bar
Развернуть
Не совместимо с Old Search (см. выше).
Оптимизирует строку поиска для быстрого выбора поисковой системы.

Progress Indicator
Развернуть
Progress Indicator добавляет на панель инструментов индикатор загрузки страницы. Может использоваться вместо растягивающегося интервала.

Tabs Focus Без перезапуска
Развернуть
Tabs Focus переключает вкладки при наведении мыши если не включен предварительный просмотр...

Style CSS Без перезапуска
Развернуть
Style CSS расширение для изменения стилей. Для создания правил CSS откройте папку style_css_files которая находится в каталоге профиля
или используйте для открытия кнопку расширения.
Для подключения файлов в каталоге используйте chrome://style_css_files/content/*
Для изменения интерфейса браузера на определённой странице используйте в начале такой код #main-window[stylecssurl="URL адрес полностью"]
или #main-window[stylecssurl^="префикс URL адреса"]

User Agent and other buttons Без перезапуска
Развернуть
Юзер агент, читать почту, запуск приложений с аргументами и др...
Разделяйте аргументы запятыми, если используется более одного аргумента для запуска приложения,
пример аргументов для запуска SeaMonkey Mail -mail,-P,default .
Можно использовать специальный аргумент OpenCurrentURI = адрес текущей страницы,
или так для Edge microsoft-edge:OpenCurrentURI = microsoft-edge: + адрес текущей страницы.
