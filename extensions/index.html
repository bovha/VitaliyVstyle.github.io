Adds Toolbar and Buttons
Развернуть
Adds Toolbar and Buttons добавляет панель инструментов, кнопки и пункты в меню для перезагрузки.
Вы найдете кнопки, для управления скрытыми функциями или требующими слишком много кликов, но некоторые функции добавляет данное расширение.

Long Left Click Без перезапуска
Развернуть
Long Left Click открывает ссылки и закладки в новой вкладке "долгим" кликом.

Agent Sheet Без перезапуска
Развернуть
Agent Sheet изменяет скроллбар и всплывающие подсказки.

Control Buttons
Развернуть
ОС: Windows, Linux.
Control Buttons добавляет кнопки (с дополнительными функциями) управления окном.
В SeaMonkey (Windows, Linux), Firefox (Linux), Pale Moon (Linux) скрывает / заменяет заголовок окна.

eCleaner
Развернуть 
eCleaner очищает настройки расширений оставленные после удаления, а также мусорных файлов в профиле и т. д.
Исходный код был взят из Preferences Monitor и немало изменён, также добавлена кнопка для панели инструментов и русская локаль.

Функции кнопок в окне eCleaner:
"Выбрать" - выбрать все видимые элементы, не отменяя выделение других элементов, которые уже выбраны.
"Выбрать все" - выбрать все элементы в текушей вкладке.
"Инвертировать" - изменение выбранного состояния всех элементов. Выбранные элементы становятся невыбранными,
                  и невыбранные элементы становятся выбранными.
"Удалить" - удалить выбранные элементы.

Old search Без перезапуска
Развернуть
Не совместимо с Quick Search Bar (см. ниже).

Развернуть

Progress Indicator
Развернуть
Progress Indicator добавляет на панель инструментов индикатор загрузки страницы. Может использоваться вместо растягивающегося интервала.
Для изменения цвета индикатора и текста можно использовать такой стиль: 

#b-statusbar-progresspanel #b-statusbar-stack #b-statusbaricon .progress-bar {
    background: rgb(242,122,0) !important;
}

#b-statusbar-progresspanel #b-statusbar-stack #b-statusbaricon .progress-remainder {
    background: rgba(0,0,0,0.4) !important;
}

#b-statusbar-progresspanel[progre="true"] #b-statusbar-stack #before-statusbar-text,
#b-statusbar-progresspanel[progre="true"] #b-statusbar-stack #after-statusbar-text {
    color: white !important;
}

Style CSS
Развернуть
Style CSS расширение для изменения стилей. Для создания правил CSS откройте папку style_css которая находится в каталоге профиля
или используйте для открытия кнопку расширения.
Для изменения интерфейса браузера на определённой странице используйте в начале такой код #main-window[stylecssurl="URL адрес полностью"]
или #main-window[stylecssurl^="префикс URL адреса"]

User Agent and other buttons Без перезапуска
Развернуть
Юзер агент, читать почту, запуск приложений с аргументами и др...
Разделяйте аргументы запятыми, если используется более одного аргумента для запуска приложения,
пример аргументов для запуска SeaMonkey Mail -no-remote,-mail,-P,default .
Можно использовать специальный аргумент OpenCurrentURI = адрес текущей страницы,
или так для Edge microsoft-edge:OpenCurrentURI = microsoft-edge: + адрес текущей страницы.
