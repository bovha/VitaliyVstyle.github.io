FULL STYLE
Update: 23-09-2017

Описание: Стиль для моего расширения Style CSS
Содержимое архива распаковать в папку профиля Pale Moon.
Использовать данный стиль с темой по умолчанию или с лёгкими темами (если тема тёмная со светлым текстом
будет использоваться оформление от стиля как на скриншотах).
Содержимое файла stylecss_author.css почти не нужно для интерфейса и можно удалить.

Стиль не простой и не предназначен для всех...

Можно установить расширение Control Buttons для замены системных кнопок управления окном,
а также чтобы убрать заголовок окна в Linux.

Автор: Vitaliy V.
Браузер: Pale Moon 27 +
ОС: Windows, Linux





При клике по кнопке меню открывается собственное меню,
а при наведении курсора на левую часть кнопки или нажатии клавиш Alt, F10 появляется классическое меню.


Боковая панель (если включена) появляется при наведении на левый край (2px в развёрнутом окне и 5px в обычном).
