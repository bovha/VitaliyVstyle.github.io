Adds Toolbar and Buttons добавляет панель инструментов, кнопки, растягивающие интервалы, интервалы, разделители и пункты в меню для перезагрузки. Вы найдете кнопки, для управления скрытыми функциями или требующими слишком много кликов, но некоторые функции добавляет данное расширение.

Без перезапуска
Long Left Click открывает ссылки и закладки в новой вкладке "долгим" кликом.

Без перезапуска
Agent Sheet изменяет скроллбар и всплывающие подсказки.

ОС: Windows, Linux.
Control Buttons добавляет кнопки (с дополнительными функциями) управления окном. В SeaMonkey и в Firefox (Linux) скрывает / заменяет заголовок окна.

eCleaner очищает настройки расширений оставленные после удаления, а также мусорных файлов в профиле и т. д. Исходный код взят из Preferences Monitor с некоторыми изменениями, добавлена кнопка для панели инструментов и русская локаль.

Без перезапуска
Right click close tab позволяет закрывать вкладки правой кнопкой мыши. Для отмены удерживайте клавишу Shift + ПКМ чтобы появилось контекстное меню вкладки.

Flip Close Tab переключает фокус на последнюю посещённую вкладку при закрытии текущей или при клике по кнопке расширения.

Без перезапуска
Не совместимо с Quick Search Bar (см. ниже).
Old Search возвращает старый способ поиска (в FF 43 и выше) для панели поиска. И добавляет новые функции: индикатор добавления поиска, опция очистить форму после поиска и др.

Не совместимо с Old Search (см. выше).
Оптимизирует строку поиска для быстрого выбора поисковой системы.

Progress Indicator добавляет на панель инструментов индикатор загрузки страницы. Может использоваться вместо растягивающегося интервала.
Для изменения цвета индикатора можно использовать такой стиль: 

#b-statusbar-progresspanel #b-statusbaricon {
    border-color: rgb(242,122,0) !important;
}

#b-statusbar-progresspanel #b-statusbaricon .progress-bar {
    background: rgb(242,122,0) !important;
}

Без перезапуска
Tabs Focus переключает вкладки при наведении мыши если не включен предварительный просмотр...

Style CSS расширение для изменения стилей. Для создания правил CSS откройте папку style_css которая находится в каталоге профиля
или используйте для открытия кнопку расширения.
Для изменения интерфейса браузера на определённой странице используйте в начале такой код #main-window[stylecssurl="URL адрес полностью"]
или #main-window[stylecssurl^="префикс URL адреса"]

Без перезапуска
Юзер агент, читать почту, запуск приложений с аргументами и др...
Разделяйте аргументы запятыми, если используется более одного аргумента для запуска приложения,
пример аргументов для запуска SeaMonkey Mail -no-remote,-mail,-P,default
